## 3. Wave 2 plan (not yet started — begin here if resuming after Wave 1 lands)

1. Read the five candidate files in `kb/maxwell/` (and the ≤25-line summaries if the session transcript survives).
2. **Judge pass** (Fable if available; else one Opus judge with this section as its brief): score candidates against C1–C6; identify complementarities — the expected outcome is NOT one winner but a composition (working prediction, to be tested, not assumed: E as reference semantics, A as engine shape, B as theorem frame, D as the metacircular spec/Language seed, C as a federation/contested lens if it survives its own refutation clause).
3. **Unification round** (one Opus agent): draft THE page — the actual candidate Maxwell equations — from the judged composition, with the losers' best organs grafted and every line tagged.
4. **The bet** (Kay protocol): Govert names the implementer (human or agent), the deadline (one week), and the wager; the page is frozen; the implementation attempt is the experiment. Divergences between page and implementation are findings, recorded.
5. Mechanization spike (D22) attaches to whichever theorems the winning page makes cheapest — determinism and monotone degradation first.
