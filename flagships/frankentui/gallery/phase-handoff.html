A template is a named, versioned prompt whose method envelope rides every artifact it produces — so “which method improves and which regresses” is a query over the ledger, not an anecdote. Changing a prompt is a version bump and a new file; the old one stays because old iteration records cite it.

The two divergence leads — it1's most valuable output

Files that cite their Rust source in a header but do not substantively port it. A worker refuted each; an independent checker upheld the refutation against the same two files. No grep would have found these — they look ported until you read them.

alpha_investing → AlphaInvestor simplified reimpl

src/FrankenTui.Runtime/AlphaInvesting.cs — 44 lines vs Rust's 562
  • alpha strategy differs: Rust scales wealth by investment_fraction (0.1); C# uses fixed DefaultAlpha (0.05) clamped.
  • FDR fallbacks (bonferroni_test, benjamini_hochberg) omitted entirely.
  • RewardFactor 1.0 vs 0.5 — doubles wealth replenishment; no p-value clamping; no tests.

schedule_trace → ScheduleTrace incompatible port

src/FrankenTui.Runtime/ScheduleTrace.cs
  • checksum incompatible: Rust byte-discriminants + FNV-1a vs C# GetHashCode() — same events, different checksums.
  • convenience API absent: no spawn/start/complete/cancel, only Record().
  • all sampling/comparison infra dropped (VoiSampler, summary(), IsomorphismProof).

These become the first entries in A2's divergence queue. Whether each is an intentional divergence is A2's call, on the record — these are the leads, not the verdict.

Stop rules are explicit: any fabricated evidence halts the triple; acceptance <60% or overturn >25% means the worker template is broken (version-bump, don't grind); cost >2× prior means a dispatcher problem. Template retirements and tier escalations escalate to the steward.

What isn't fixed — recorded, not hidden

Target resolution 96.2% degraded. Per-project Roslyn without NuGet reference resolution can't discriminate a repo that genuinely doesn't compile in isolation. Track B #1. Until fixed, “degraded” is near-constant and not a useful signal.
Full-map Atlas is 31.3 MB against a 15 MB budget — a standing breach. Data-island compaction is Track B #2, promoted before the gallery grows further.
The name-norm queue (1,632) is not earned. Its structural false-positive mode (generic test-fixture names) is proven; it waits for a pre-filter or a better heuristic — not a wholesale drain.
Provenance pin ~2 months stale; 131 impl-block identity duplicates. Both surfaced, both queued in ATTIC with triggers. Nothing silently dropped.
